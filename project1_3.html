To get TOP and CRD with tLeap program of AMBER

AMBER requires three files to run. Two of these will be prepared in this step. The first is a topology file (*.top/*.prmtop), which contains all the information about the force field, including bonds, angles, torsions, charges, van der Walls, and atom types. The second is a coordinate file (*.crd/*.inpcrd), which contains the positions of all the atoms in Cartesian coordinate space. The tLeap program will be used to generate topology and coordinate files of your homology model from its pdb.

The top portion of the PDB file contains the protein information, with the counterion and solvent information following below. Each of these sections is separated by a TER card. The fifth column of the file gives the residue numbers. Type /WAT to use the vi editor search command to jump down to the first occurence of a water molecule in the file. Here you can see where the protein residue numbers end and water residue numbers begin. Type :$ to travel to the end of the file and get the final water residue number. Record your residue ranges.
